Sabes que o pinheiro também dá fruto? São os pinhões, que estão dentro das pinhas e que sabem muito bem!

Os pinheiros têm mais utilidade do que apenas para árvore de Natal! Para além do saboroso fruto que se encontra dentro das pinhas (os pinhões), os pinheiros também são fonte de madeira para construir barcos e para usar na lareira.

Eu adoro compota de figo com pinhões! Até fico com água no bico!


Para lembrar


1 - O fruto dos pinheiros que se encontram nas pinhas são os pinhões;

2 - A madeira do pinheiro é usada para construir barcos e para lenha.
