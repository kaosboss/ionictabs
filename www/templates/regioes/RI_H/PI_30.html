As abóboras não servem apenas para fazer máscaras! São muito boas para as sopas e para as compotas! As amoras são também muito boas para compotas!

Aposto que já viste máscaras de Halloween feitas de abóbora! Mas a abóbora é um fruto muito bom para fazer ricas sopas e compotas.

A amoreira dá-nos as lindas amoras que têm um sabor e uma textura únicas! Alem das amoras as amoreiras têm umas folhas muito apetitosas para os bichos-da-seda que se alimentam delas.


Para lembrar


1 - A abóbora é um fruto;

2 - As abóboras são muito boas para fazer sopas ou compotas;

3 - As folhas da amoreira são utilizadas para alimentar os bichos-da-seda.
